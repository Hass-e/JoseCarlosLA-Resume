About me

I’m an Electronic Engineer from the Autonomous University of Baja California and a QA Engineer with experience in both manual and automated testing. I value manual testing for its attention to detail and deep understanding of product behavior, while automation allows me to improve efficiency, reliability, and test coverage.

I build automated tests for web applications using WebdriverIO, Cypress, and Selenium with JavaScript/TypeScript and/or Python, manage code with Git/GitHub, and integrate tests into Jenkins CI/CD pipelines. I also have a solid understanding of HTML and CSS, and previously developed an automated Excel tool with VBA to improve team workflows.

I enjoy collaborating with others, building strong team relationships, and continuously learning. I’m seeking opportunities where I can grow, add value, and contribute to a culture that values quality and continuous improvement.
